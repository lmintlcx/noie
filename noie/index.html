IE 浏览器早已过时

Internet Explorer 伴随着微软的操作系统而发布, 没有积极地迭代更新以适应互联网的飞速发展, 不仅不支持最新的网页文档格式标准, 而且早已年久失修充满了各种安全漏洞. 微软已经宣布不再为 IE 浏览器 11 以下的版本提供相应的支持, 即便是最新版本的 IE 11 也是早在 2013 年发布的, 相比之下其他主流浏览器 (Chrome / Firefox) 则是一直在持续着维护和更新.

共同抵制 IE 浏览器

为了兼容这些不成熟的史前软件, 网页设计人员需要付出大量的额外的代码工作, 受制于远古技术条件最终实现的效果也往往做不到令人满意. 过于老旧的没有更新的软件同时也是一个潜在的安全隐患, 使用户暴露在容易受到有害病毒和恶意软件攻击的环境中, 造成业务数据受损或失窃等危害. 有鉴于此, 请和我们一起抛弃这些过时的浏览器版本 (IE 11/10/9/8/7/6/...).

更先进的现代浏览器

为了保证数据安全和最佳浏览体验, 我们推荐使用这些现代浏览器的最新版本来访问互联网.

  • 谷歌浏览器 Google Chrome
  • 火狐浏览器 Mozilla Firefox
  • 边缘浏览器 Microsoft Edge
  • 欧朋浏览器 Opera
